# Quick Start Project
A quick set-up for creating the front-end of a website.

## Instructions


### Frameworks
* Animate JS
* Bootstrap v4
* Font Awesome
* Google Fonts
* Jquery 3.2.1
* Popper.js

### Preprocessor
* HAML
* SASS

### REQUIREMENTS
**Ruby 2.0+**
- Gems Required:   
    - **Haml** 
    > sudo gem install haml
    
    - **Observr**
    > sudo gem install observr

    - **Sass**
    > sudo gem install sass --no-user-install'

    - **Watchr**
    > sudo gem install watchr
